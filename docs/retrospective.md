# Retrospectiva final

## Qué aprendí
Durante este proyecto conecté todo lo aprendido en las fases anteriores:
React con TypeScript para el frontend, Express para el backend y Tailwind
para los estilos. Aprendí a estructurar una app fullstack con capas claras
y tipos alineados entre frontend y backend.

## Principales problemas
- Configurar CORS entre frontend y backend en producción
- Gestionar variables de entorno entre Vercel y Railway
- Los errores de TypeScript en el build de Vercel por imports de tipos

## Cómo usé la IA
Usé IA para guiarme en la estructura del proyecto, resolver errores de
TypeScript y configurar el despliegue en Vercel y Railway.

## Reflexión final
Desarrollar una app fullstack completa me ha dado una visión global del
desarrollo web profesional. La combinación de TypeScript en frontend y
backend reduce drásticamente los errores en producción.
