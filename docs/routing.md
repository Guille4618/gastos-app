# Rutas

## Estructura
- `/` → Página principal con formulario, filtros y lista de gastos
- `/estadisticas` → Página de estadísticas por categoría
- `*` → Página 404 para rutas no encontradas

## Implementación
Se usa React Router v6 con `BrowserRouter`, `Routes` y `Route`.
La navegación entre páginas se hace con el componente `Link`.
