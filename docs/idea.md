# Idea del proyecto: GastosApp

## ¿Qué problema resuelve?
Muchas personas no saben en qué gastan su dinero cada mes. GastosApp
permite registrar gastos de forma rápida, categorizarlos y ver estadísticas
para tomar mejores decisiones financieras.

## Usuario objetivo
Cualquier persona que quiera llevar un control de sus gastos personales
sin necesidad de usar hojas de cálculo complejas.

## Funcionalidades principales
- Añadir un gasto con nombre, cantidad, categoría y fecha
- Ver el listado de todos los gastos
- Ver el total gastado por categoría
- Eliminar un gasto
- Filtrar gastos por categoría o fecha

## Funcionalidades opcionales
- Editar un gasto existente
- Exportar los gastos a CSV
- Establecer un presupuesto mensual por categoría
- Gráfico visual de gastos por categoría

## Mejoras futuras
- Autenticación de usuarios para tener datos privados
- Soporte para múltiples monedas
- Notificaciones cuando se supera el presupuesto
- App móvil con React Native

## Tecnologías
- Frontend: React + TypeScript + Tailwind CSS + Vite
- Backend: Node.js + Express
- Datos: API REST propia con datos en memoria o LocalStorage
